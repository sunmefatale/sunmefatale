저는 **기획에 강한 데이터 분석가**를 목표로 성장하고 있습니다.  
데이터는 단순한 숫자가 아니라, **의미와 스토리**라고 믿습니다.  
그래서 **Python · Pandas · Visualization**을 활용해 데이터를 해석하고,  
철학과 심리학적 관점까지 더해 **사람을 이해하는 분석**을 지향합니다. 🍹
